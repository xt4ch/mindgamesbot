Для работы бота, необходимо установить NodeJS (18.7.0 Current или новее)
https://nodejs.org/en/
==================================================================================
1. Установить NodeJS 18.7.0 и выше (https://nodejs.org/en/)
2. Откройте файл .env, в нем укажите "GAME_SERVER_IP=" - IP игрового сервера,
GAME_SERVER_PORT= - Порт игрового сервера, 
STEAM_API_KEY= - API ключ steam (https://steamcommunity.com/dev/apikey),
DISCORD_TOKEN= - Токен discord бота (https://discord.com/developers/applications).
3. Запустите файл install.bat (только при первом запуске на машине).
4. Запустить файл start.bat.
